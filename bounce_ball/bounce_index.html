Instructions for the Game

What is this?

This is a fun game in which you have to eat all balls. There is no time limit. Press the restart button to play again and have fun.

How to play?

Key Functions

    Movement

  • W - Move Upwards
  • A - Move to Left
  • S - Move Downwards
  • D - Move to Right

    • Ball Count

  • B - Add More Balls

    • Ball Size

  • Q - Increase Your Ball Size
  • E - Decrease Your Ball Size
